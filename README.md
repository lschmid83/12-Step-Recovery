# 12-Step Recovery

12-Step Recovery is an in-development Android companion app for recoverymeetingfinder.com. When it is finished it aims to provide 12-Step reading materials, daily quotes, reading material, motivational images, an in-person meeting map and motivational speaker tapes.

This repository contains the source code for the Android Studio Java project.

We are looking for contributors who want to help finish the app. There issue list outlines the features which still need work or have not been implemented yet.
